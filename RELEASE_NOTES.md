## DeepM v0.1.0

Локальный переводчик (приватный аналог DeepL) на базе Tencent HY‑MT1.5. Работает полностью офлайн — текст никуда не уходит. Перевод в окне, плавающая кнопка по выделению, перевод‑замена по горячей клавише, словарь, история.

### Что скачать

| Файл | Для кого | Размер |
|------|----------|--------|
| **DeepM_0.1.0_x64-setup.exe** | Для всех (CPU). Запускается на любом ПК с Windows 10/11 x64. | ~12 МБ |
| **DeepM-cuda-pack.zip** | Ускорение для NVIDIA — полный пакет, ни от чего не зависит. | ~764 МБ |
| **DeepM-cuda-pack-slim.zip** | Ускорение для NVIDIA, **если у вас уже установлен CUDA Toolkit 12.x**. | ~307 МБ |

> ⚠️ Приложение само определяет вашу видеокарту: в Настройках галочка «Использовать GPU» активна только если ускорение реально заработает. Если NVIDIA нет — галочка погашена.

### Установка

1. Скачайте и запустите **DeepM_0.1.0_x64-setup.exe**. Если Windows покажет SmartScreen — «Подробнее» → «Выполнить в любом случае» (приложение пока без подписи).
2. При первом запуске откройте **Менеджер моделей** и скачайте модель (рекомендую начать с **1.8B Q4_K_M**, ~1.1 ГБ). Модель качается один раз и хранится локально.
3. Готово — выделяйте текст и переводите.

### Ускорение на NVIDIA (необязательно)

Если у вас видеокарта NVIDIA и свежие драйверы:
1. Скачайте **DeepM-cuda-pack.zip** (полный — подойдёт всем с NVIDIA). Либо, если у вас уже стоит **CUDA Toolkit 12.x**, возьмите лёгкий **DeepM-cuda-pack-slim.zip** — недостающие библиотеки `cublas`/`cudart` подхватятся из системы.
2. Распакуйте папку `engine` из архива в каталог установки DeepM (по умолчанию `C:\Program Files\DeepM\` или `%LOCALAPPDATA%\Programs\DeepM\`), подтвердив замену/добавление файлов.
3. Перезапустите DeepM. В Настройках галочка «Использовать GPU» станет активной — включите её и нажмите «Перезагрузить движок». Перевод пойдёт на GPU и станет заметно быстрее (особенно на 7B).

**Почему `ggml-cuda.dll` нельзя не качать:** это CUDA-бэкенд самого llama.cpp (≈540 МБ), его нет в системной установке CUDA. А вот `cublas`/`cublasLt`/`cudart` (≈547 МБ) могут браться из CUDA Toolkit — поэтому для них и есть slim-вариант.

### Требования

- Windows 10/11, 64‑бит.
- WebView2 — установится автоматически, если отсутствует.
- Модель скачивается отдельно внутри приложения (не входит в установщик).
