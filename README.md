# DeepM

Локальный переводчик для Windows — приватная альтернатива DeepL, работающая полностью на вашем устройстве. Использует модель [Tencent HY-MT1.5](https://huggingface.co/hkunlp/hy-mt) через [llama.cpp](https://github.com/ggerganov/llama.cpp).

**Все переводы остаются на вашем устройстве. Интернет нужен только для первоначальной загрузки модели.**

---

## Возможности

- **Панель перевода** — двухпанельный интерфейс, drag-to-resize разделитель, обмен языков
- **Плавающая кнопка** — выделите текст в любом приложении, через ~1 секунду появится кнопка перевода
- **Translate & Replace** — горячая клавиша для перевода выделенного текста прямо в нём
- **Тройное Ctrl+C** — нажмите Ctrl+C три раза быстро чтобы отправить текст в переводчик
- **История переводов** — хранится локально
- **Глоссарий** — пользовательские термины, которые передаются модели
- **GPU-ускорение** — поддержка CUDA (рекомендуется RTX/GeForce)
- **Системный трей** — работает в фоне, открывается двойным кликом
- **Автозапуск** — опциональный старт вместе с Windows

---

## Требования

- Windows 10/11 (x64)
- [Node.js 18+](https://nodejs.org/)
- [Rust toolchain](https://rustup.rs/)
- Для GPU: видеокарта NVIDIA с драйверами CUDA 12.x

## Установка

### 1. Клонировать и установить зависимости

```bash
git clone https://github.com/mikheys/DeepM.git
cd DeepM
npm install
```

### 2. Скачать llama-server

1. Перейдите на страницу [релизов llama.cpp](https://github.com/ggerganov/llama.cpp/releases/latest)
2. Скачайте `llama-b*-bin-win-cuda-12.4-x64.zip` (с GPU) или `llama-b*-bin-win-x64.zip` (CPU)
3. Распакуйте **все файлы** в папку `src-tauri/binaries/`

> ⚠️ `llama-server.exe` требует сопутствующих DLL в той же папке — распакуйте весь архив, не только exe.

### 3. Запуск в режиме разработки

```bash
npm run tauri dev
```

При первом запуске перейдите во вкладку **Models** и скачайте модель HY-MT1.5 (~1.1 ГБ для варианта Q4_K_M).

### 4. Сборка релиза

```bash
npm run tauri build
```

## Архитектура

```
DeepM/
├── src/                      # React + TypeScript (фронтенд)
│   ├── components/           # UI-компоненты
│   ├── styles/               # Глобальный CSS + CSS-переменные
│   └── api.ts                # Tauri IPC биндинги
└── src-tauri/                # Rust (бэкенд)
    ├── src/
    │   ├── core/             # Движок перевода, менеджер моделей, история
    │   ├── os_integration/   # Трей, плавающая кнопка, буфер обмена, хоткеи
    │   ├── shortcuts/        # Конфигурация горячих клавиш
    │   └── lib.rs            # Точка входа + Tauri-команды
    └── binaries/             # llama-server.exe + DLL (не в git)
```

**Стек:** Tauri 2 · React 18 · TypeScript · Vite 6 · Rust · llama.cpp

## Модель

HY-MT1.5 — многоязычная модель перевода от Tencent, оптимизированная для китайского, английского, русского и других языков. Скачивается в формате GGUF через интерфейс приложения.

| Вариант       | Качество          | VRAM / RAM |
|---------------|-------------------|------------|
| 1.8B Q4_K_M   | Хорошее, быстрое  | ~1 ГБ      |
| 1.8B Q8_0     | Лучше             | ~1.9 ГБ    |
| 7B Q4_K_M     | Наилучшее         | ~5 ГБ      |
